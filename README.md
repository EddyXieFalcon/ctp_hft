QT做界面，使用ctp api实现一个快速急速交易系统

# 编译环境
windows7 + vs2013

# qt版本
qt-opensource-windows-x86-msvc2013-5.5.1.exe
qt-vs-addin-1.2.4-opensource.exe

# ctp api版本
windows32生产版本tradeapi(V6.3.5)
http://www.simnow.com.cn/static/softwareDownload.action
